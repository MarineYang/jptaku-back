# 온보딩 데이터 명세 (Onboarding Data Specification)

이 문서는 `src/pages/Onboarding.tsx`에 정의된 온보딩 프로세스의 모든 선택 요소들을 정리한 파일입니다. 백엔드 DB 설계 및 API 응답 데이터 구성 시 참고할 수 있습니다.

---

## 1. 관심사 대분류 (Categories)

사용자가 가장 먼저 선택하는 주요 관심사입니다.

| ID | Label | Icon (Lucide React) |
| :--- | :--- | :--- |
| `anime` | 애니/만화 | Tv |
| `game` | 게임 | Gamepad2 |
| `music` | 음악/Jpop/버튜버 | Music |
| `lifestyle` | 오타쿠 라이프스타일 | ShoppingBag |
| `situation` | 실전 오타쿠 상황 | MessageCircle |

---

## 2. 상세 장르 (Sub Categories)

대분류(`Category ID`) 선택 시 노출되는 하위 선택지입니다.

### 📺 anime (애니/만화)
- 이세계/판타지
- 러브코미디
- 일상물
- 배틀/액션
- 스포츠물
- SF/로봇
- 음악/아이돌물
- 미스터리/추리

### 🎮 game (게임)
- JRPG
- 모바일 가챠게임
- 리듬게임
- FPS
- 닌텐도 게임
- 격투 게임

### 🎵 music (음악/Jpop/버튜버)
- Jpop
- Vocaloid
- 애니송
- 아이돌
- 버튜버(hololive, NIJISANJI 등)

### 🛍️ lifestyle (오타쿠 라이프스타일)
- 성지순례
- 굿즈 구매
- 피규어/프라모델
- 코미케/행사 참가
- 애니카페 방문
- 게임센터 방문

### 💬 situation (실전 오타쿠 상황)
- 굿즈 예약하기
- 행사에서 인사하기
- 친구와 애니 얘기하기
- 일본 사이트 주문하기
- 일본 여행 오타쿠 코스
- 콘서트/라이브 관람

---

## 3. 일본어 레벨 (Levels)

사용자의 현재 일본어 구사 능력입니다.

| ID | Label | Description (상세 설명) | 대응 JLPT (추정) |
| :--- | :--- | :--- | :--- |
| `lv0` | Lv 0 - 완전 초입문 | 히라가나/가타카나도 모름 | - |
| `lv1` | Lv 1 - 기본 인사 가능 | N5 수준 | N5 |
| `lv2` | Lv 2 - 일상 회화 조금 가능 | N4 수준 | N4 |
| `lv3` | Lv 3 - 생각 표현 가능 | N3 수준 | N3 |
| `lv4` | Lv 4 - 능숙 | N2 수준 | N2 |
| `lv5` | Lv 5 - 거의 원어민 수준 | N1 수준 | N1 |

---

## 4. 학습 목적 (Purposes)

사용자가 일본어를 배우려는 주된 이유입니다. (다중 선택 가능)

- 자막 없이 애니·만화 즐기려고
- 일본 친구와 대화하고 싶어서
- 일본 여행에서 말하고 싶어서
- 버튜버 방송/콘텐츠 이해하고 싶어서
- 좋아하는 게임의 일본 서버/콘텐츠 즐기려고
- 굿즈 구매·이벤트 참가 때문에
- 기타
